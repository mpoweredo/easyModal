Simplify your modal.

Save time!

Don't spend a lot of time creating good looking modal.
easyModal let you make modal in one line of code.

new EasyModal({title: 'France', description: 'The capital city of France is Paris.', theme: 'dark'});

Install via npm!

easyModal is available on npm and can be
installed from the command line via npm

npm install easy-modal-js --save

easyModal is designed to be easy to use, with one life of code we can create nice looking modal with simple configuration.

You can also change styles of modal using modal.scss file.

Here's a line of code representing a simple information modal:

new EasyModal({title: 'France', description: 'The capital city of France is Paris.', theme: 'dark'});
